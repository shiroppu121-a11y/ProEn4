# 2D横スクロールアクションゲーム

## 1. ソースコードの構成

本プログラムは、PythonとPygameを用いて作成した2D横スクロールアクションゲームである。

主なファイル構成は以下の通りである。

```text
ProEn4/
│
├─ main.py
├─ player.py
├─ enemy.py
├─ item.py
├─ bullet.py
├─ update.py
├─ draw.py
├─ game_state.py
├─ map.py
├─ settings.py
├─ records.py
├─ assets.py
│
└─ assets/
    ├─ background.png
    ├─ player/
    │   └─ player.png
    └─ ...
```

### 各ファイルの役割

| ファイル名           | 役割                                 |
| --------------- | ---------------------------------- |
| `main.py`       | ゲーム全体の実行、画面遷移、キー入力の処理              |
| `player.py`     | プレイヤーの移動、ジャンプ、アイテム使用などの処理          |
| `enemy.py`      | 敵の生成、移動、描画などの処理                    |
| `item.py`       | アイテムの生成、取得、アイテム種類の管理               |
| `bullet.py`     | プレイヤーが使用する投擲アイテムの処理                |
| `update.py`     | ゲーム中の状態更新、衝突判定、ゴール判定               |
| `draw.py`       | タイトル画面、ゲーム画面、ポーズ画面、クリア画面、記録画面などの描画 |
| `game_state.py` | ゲームの状態やタイマーなどの管理                   |
| `map.py`        | ステージの足場や地形の管理                      |
| `settings.py`   | 画面サイズ、ゲーム速度、ゴール位置などの設定             |
| `records.py`    | クリアタイムの保存と読み込み                     |
| `assets.py`     | 画像などのアセットの読み込み                     |
| `assets/`       | プレイヤーや背景などの画像ファイル                  |

## 2. プログラムの起動方法

### 必要な環境

* Python 3.x
* Pygame

### Pygameのインストール

ターミナルまたはPowerShellで以下のコマンドを実行する。

```bash
pip install pygame
```

### ゲームの起動

プロジェクトのフォルダを開き、以下のコマンドを実行する。

```bash
python main.py
```

または、Visual Studio Codeで`main.py`を開き、実行ボタンから起動する。

## 3. プログラムの操作方法

### タイトル画面

| キー    | 操作           |
| ----- | ------------ |
| Enter | ゲーム開始        |
| H     | 過去のクリアタイムを表示 |

### ゲーム中

| キー  | 操作          |
| --- | ----------- |
| A   | 左に移動        |
| D   | 右に移動        |
| W   | ジャンプ        |
| E   | 取得したアイテムを使用 |
| Esc | ポーズ画面を表示    |

### アイテム

ステージ上に配置されたアイテムを取得すると、画面右上に所持しているアイテムが表示される。

アイテムは以下の種類がある。

* **Speed**：プレイヤーの移動速度を上昇させる
* **Invincible**：一定時間、敵に接触してもゲームオーバーにならない
* **Throw**：弾を投げて敵を倒す

アイテムを取得した状態で`E`キーを押すと、所持しているアイテムを使用する。

### ポーズ画面

`Esc`キーを押すとゲームを一時停止できる。

ポーズ中はプレイヤーや敵の動作が停止する。

再度`Esc`キーを押すことでゲームを再開できる。

### クリア

ステージの右側にあるゴールに到達するとゲームクリアとなる。

クリア時には、

* クリアタイム
* ベストタイム

が表示される。

クリアタイムは保存され、タイトル画面から過去の記録を確認できる。

### ゲームオーバー

敵に接触するとゲームオーバーとなる。

ゲームオーバー後は以下のキーを使用できる。

| キー | 操作        |
| -- | --------- |
| R  | ゲームをリスタート |
| T  | タイトル画面に戻る |

### クリア後

クリア画面では以下の操作ができる。

| キー | 操作        |
| -- | --------- |
| R  | ゲームをリスタート |
| T  | タイトル画面に戻る |

---

## 4. ゲームの目的

プレイヤーを操作してステージを進み、敵や障害物を避けながらゴールを目指す。

ステージ上に配置されたアイテムを取得して使用することで、移動速度の上昇、一定時間の無敵、敵への攻撃などの効果を得ることができる。

また、クリアタイムは記録されるため、より短い時間でのクリアを目指すことができる。
